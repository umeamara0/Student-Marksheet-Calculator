Student Marksheet Calculator
A dynamic web application built with HTML, CSS, and JavaScript that calculates total marks, percentages, and grades for students.

 Features
- Real-time calculation logic.
- Grade assignment based on percentage.

 Technologies Used
- **HTML5**: For semantic structure.
- **CSS3**: For styling and layout.
- **JavaScript (ES6)**: For calculation logic and DOM manipulation.
